Terminos Vistos en clase

funcion
La función, es basicamente una acción o un procedimiento que va a realizar desde un arreglo o una variable, este debe tener una entrada y una salida.
let
Es una palabra que ayuda a declarar una variable, cuyo valor puede cambiar conforme se avance el codigo.
const
Const al igual que let es un apoyo para declarar una variable su diferencia con let, es que esta no cambiara su valor y quedara guadado para seguir usandose mas adelante.
output
Es un contenedor en el que el sitio o la aplicacion puede inyectar los resultados de un calculo o resultado de una accion.
video
Se utiliza para agrafar videos en un documento HTML o XHTML.
embed
Es un punto de integracion para una aplicacion externa o contenido interactivo
